# 🎬 VidNova

A modern Full Stack Video Streaming Platform inspired by Netflix and YouTube.

VidNova allows users to discover trending content, watch videos, upload videos, interact with creators, and enjoy a seamless streaming experience through a modern dark-themed interface.

> 🚧 Project Status: Under Development

---

## 🚀 Features Implemented

### Frontend

* Modern Netflix-style Homepage
* Responsive Dark UI
* Navigation Bar
* Hero Section
* Search Bar UI
* Featured Video Section
* Trending Videos Grid
* Categories Section
* Video Watch Page
* HTML5 Video Player
* Related Videos Sidebar
* Creator Information Section
* Subscribe Button UI

* React Router Navigation

* Express.js Server Setup

## ⚡ Installation

### Clone Repository

```bash
git clone https://github.com/karankumar-tech-eng/VidNova.git
```

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm run dev
```
